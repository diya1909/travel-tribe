# Travel Tribe 🌍

Travel Tribe is a web platform where travelers can share their travel stories, connect with other travelers, and explore travel experiences.

## Project Structure

Frontend: travel-tribe  
Backend: traveltribe

### Frontend
Handles the user interface including:
- User login/signup
- Profile page
- Travel stories
- Map integration

### Backend
Handles server-side logic including:
- API routes
- Authentication
- Database operations

## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express

## How to Run

Frontend:
cd travel-tribe
npm install
npm start


Backend:

cd traveltribe
npm install
node server.js
